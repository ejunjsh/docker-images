# Build elasticsearh image
```
sh build-image.sh
```

# Run elasticsearh cluster

## Run elasticsearh master
```
sh start-master.sh
``` 

## Run elasticsearh slave, you can pass different slave-id to create more slaves.
```
sh start-slave.sh <slave-id>
```

## Run echead
```
sh start-echead.sh
```
